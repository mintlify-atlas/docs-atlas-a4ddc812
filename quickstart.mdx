This quick start guide will help you install Mage and create your first data pipeline.

## Installation

Choose your preferred installation method:

<CodeGroup>
```bash pip
pip install mage-ai
```

```bash Docker
docker pull mageai/mageai:latest
docker run -it -p 6789:6789 -v $(pwd):/home/src mageai/mageai mage start demo_project
```

```bash conda
conda install -c conda-forge mage-ai
```
</CodeGroup>

<Note>
Mage requires Python 3.9 or higher. Check your Python version with `python --version`.
</Note>

## Create your first project

<Steps>
  <Step title="Initialize a new project">
    Create a new Mage project in your desired directory:
    
    ```bash
    mage init my_first_project
    cd my_first_project
    ```
    
    This creates a project with the following structure:
    - `pipelines/` - Your data pipelines
    - `data_loaders/` - Blocks for loading data
    - `transformers/` - Blocks for transforming data
    - `data_exporters/` - Blocks for exporting data
    - `utils/` - Helper functions
  </Step>
  
  <Step title="Start the Mage server">
    Launch the Mage UI and server:
    
    ```bash
    mage start
    ```
    
    Open your browser and navigate to `http://localhost:6789`.
    
    <Tip>
    Customize the host and port: `mage start --host 0.0.0.0 --port 8080`
    </Tip>
  </Step>
  
  <Step title="Create a pipeline">
    In the Mage UI:
    
    1. Click **+ New pipeline**
    2. Select **Standard (batch)** as the pipeline type
    3. Name your pipeline (e.g., "my_first_pipeline")
    
    Your new pipeline will open in the editor.
  </Step>
  
  <Step title="Add a data loader">
    Add a block to load data:
    
    1. Click **+ Data loader**
    2. Select **Python** → **API**
    3. Add this code:
    
    ```python
    import requests
    
    @data_loader
    def load_data(*args, **kwargs):
        """
        Load sample data from an API
        """
        url = 'https://jsonplaceholder.typicode.com/users'
        response = requests.get(url)
        return response.json()
    ```
    
    4. Click **Run block** to test it
  </Step>
  
  <Step title="Add a transformer">
    Add a block to transform the data:
    
    1. Click **+ Transformer** → **Generic (no template)**
    2. Add this code:
    
    ```python
    @transformer
    def transform(data, *args, **kwargs):
        """
        Extract names and emails
        """
        return [{
            'name': user['name'],
            'email': user['email'],
            'company': user['company']['name']
        } for user in data]
    ```
    
    3. Click **Run block** to see the transformed data
  </Step>
  
  <Step title="Add a data exporter">
    Add a block to export the data:
    
    1. Click **+ Data exporter** → **Python** → **Local file**
    2. Add this code:
    
    ```python
    import pandas as pd
    
    @data_exporter
    def export_data(data, *args, **kwargs):
        """
        Export data to a CSV file
        """
        df = pd.DataFrame(data)
        filepath = 'output/users.csv'
        df.to_csv(filepath, index=False)
        print(f'Exported {len(df)} rows to {filepath}')
    ```
    
    3. Click **Run block** to export the file
  </Step>
  
  <Step title="Run the full pipeline">
    Execute your complete pipeline:
    
    1. Click the **Execute pipeline** button at the top
    2. Wait for all blocks to complete
    3. Check the output in your project's `output/` directory
    
    You can also run the pipeline from the command line:
    
    ```bash
    mage run my_first_project my_first_pipeline
    ```
  </Step>
</Steps>
