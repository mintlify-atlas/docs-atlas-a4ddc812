## Overview

The `Pipeline` class represents a data pipeline in Mage. It manages the execution flow of blocks, handles dependencies, and stores pipeline configuration.

## Constructor

```python
Pipeline(
    uuid: str,
    repo_path: str = None,
    config: dict = None,
    repo_config: RepoConfig = None,
    catalog: dict = None,
    context_data: dict = None,
    use_repo_path: bool = False,
    description: str = None,
    tags: list[str] = None,
    execution_framework: str = None,
)
```

<ParamField path="uuid" type="str" required>
  Unique identifier for the pipeline
</ParamField>

<ParamField path="repo_path" type="str">
  Path to the repository containing the pipeline. Defaults to current repository path.
</ParamField>

<ParamField path="config" type="dict">
  Pipeline configuration dictionary. If None, loads from metadata.yaml file.
</ParamField>

<ParamField path="repo_config" type="RepoConfig">
  Repository configuration object
</ParamField>

<ParamField path="catalog" type="dict">
  Data integration catalog configuration
</ParamField>

<ParamField path="context_data" type="dict">
  Additional context data for multi-project setups
</ParamField>

<ParamField path="use_repo_path" type="bool" default="False">
  Whether to use the provided repo_path explicitly
</ParamField>

<ParamField path="description" type="str">
  Pipeline description
</ParamField>

<ParamField path="tags" type="list[str]">
  List of tags associated with the pipeline
</ParamField>

<ParamField path="execution_framework" type="str">
  Execution framework identifier (e.g., 'argo', 'prefect')
</ParamField>

## Class Methods

### create

Creates a new pipeline with the specified configuration.

```python
@classmethod
def create(
    name: str,
    repo_path: str = None,
    description: str = None,
    pipeline_type: PipelineType = PipelineType.PYTHON,
    tags: list[str] = None,
    execution_framework: str = None,
) -> Pipeline
```

<ParamField path="name" type="str" required>
  Name of the pipeline
</ParamField>

<ParamField path="repo_path" type="str">
  Path to repository
</ParamField>

<ParamField path="description" type="str">
  Pipeline description
</ParamField>

<ParamField path="pipeline_type" type="PipelineType" default="PipelineType.PYTHON">
  Type of pipeline (PYTHON, STREAMING, INTEGRATION, DATABRICKS, PYSPARK)
</ParamField>

<ParamField path="tags" type="list[str]">
  Tags for the pipeline
</ParamField>

<ParamField path="execution_framework" type="str">
  Execution framework identifier
</ParamField>

<ResponseField name="return" type="Pipeline">
  The newly created Pipeline instance
</ResponseField>

**Example:**

```python

pipeline = Pipeline.create(
    'my_data_pipeline',

```

### get

Retrieves an existing pipeline by UUID.

```python
@classmethod
def get(
    uuid: str,
    repo_path: str = None,
    repo_config: RepoConfig = None,
    check_if_exists: bool = False,
    all_projects: bool = False,
    context_data: dict = None,
    use_repo_path: bool = False,
) -> Pipeline
```

<ParamField path="uuid" type="str" required>
  Pipeline UUID to retrieve
</ParamField>

<ParamField path="repo_path" type="str">
  Repository path
</ParamField>

<ParamField path="check_if_exists" type="bool" default="False">
  Return None if pipeline doesn't exist instead of raising an error
</ParamField>

<ResponseField name="return" type="Pipeline">
  Pipeline instance or None if not found and check_if_exists is True
</ResponseField>

**Example:**

```python
pipeline = Pipeline.get('my_data_pipeline')
print(f"Pipeline: {pipeline.name}")
print(f"Blocks: {len(pipeline.blocks_by_uuid)}")
```

## Instance Methods

### add_block

Adds a block to the pipeline.

```python

    block: Block,

    downstream_block_uuids: list[str] = None,
```

<ParamField path="block" type="Block" required>

</ParamField>

<ParamField path="upstream_block_uuids" type="list[str]">

</ParamField>

<ParamField path="downstream_block_uuids" type="list[str]">
  UUIDs of blocks that should execute after this block
</ParamField>

<ParamField path="priority" type="int">
  Execution priority order
</ParamField>

<ParamField path="widget" type="bool" default="False">
  Whether this is a widget block
</ParamField>

<ResponseField name="return" type="Block">
  The added block
</ResponseField>

**Example:**

```python
from mage_ai.data_preparation.models.block import Block

block = Block.create(
    'load_data',
    'data_loader',
    repo_path=pipeline.repo_path,
    pipeline=pipeline
)

pipeline.add_block(block, upstream_block_uuids=[])
```

```python
    global_vars=global_vars,
```

### async execute

Executes the pipeline asynchronously with parallel block execution.

```python
async def execute(
    analyze_outputs: bool = False,
    build_block_output_stdout: Callable = None,
    global_vars: dict = None,
    parallel: bool = True,
    run_sensors: bool = True,
    run_tests: bool = True,
    update_status: bool = True,
) -> None
```

<ParamField path="parallel" type="bool" default="True">
  Whether to execute independent blocks in parallel
</ParamField>

**Example:**

```python
import asyncio
from mage_ai.data_preparation.models.pipeline import Pipeline

async def run_pipeline():
    pipeline = Pipeline.get('my_pipeline')
    await pipeline.execute(
        global_vars={'env': 'production'},
        parallel=True
    )

asyncio.run(run_pipeline())
```

## Complete Example

```python
from mage_ai.data_preparation.models.pipeline import Pipeline, PipelineType
from mage_ai.data_preparation.models.block import Block

# Create a new pipeline
pipeline = Pipeline.create(
    'etl_pipeline',
    description='ETL pipeline for customer data',
    pipeline_type=PipelineType.PYTHON,
    tags=['etl', 'customers']
)

# Add blocks to the pipeline
loader = Block.create(
    'load_customers',
    'data_loader',
    repo_path=pipeline.repo_path,
    pipeline=pipeline
)

transformer = Block.create(
    'transform_data',
    'transformer',
    repo_path=pipeline.repo_path,
    pipeline=pipeline,
    upstream_block_uuids=['load_customers']
)

exporter = Block.create(
    'export_to_warehouse',
    'data_exporter',
    repo_path=pipeline.repo_path,
    pipeline=pipeline,
    upstream_block_uuids=['transform_data']
)

# Execute the pipeline
pipeline.execute_sync(
    global_vars={'environment': 'production'},
    run_tests=True
)

print(f"Pipeline {pipeline.name} executed successfully")
print(f"Total blocks: {len(pipeline.blocks_by_uuid)}")
```
