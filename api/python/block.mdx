## Overview

The `Block` class represents a single unit of code execution in a Mage pipeline. Blocks can be data loaders, transformers, exporters, sensors, or custom code.

## Constructor

```python
Block(
    name: str,
    uuid: str,
    block_type: BlockType,
    block_color: BlockColor = None,
    content: str = None,
    executor_config: dict = None,

    extension_uuid: str = None,
    groups: list[str] = None,
    status: BlockStatus = BlockStatus.NOT_EXECUTED,
    pipeline: Pipeline = None,
    replicated_block: str = None,
    retry_config: dict = None,
    language: BlockLanguage = BlockLanguage.PYTHON,
    configuration: dict = None,
    has_callback: bool = False,
    timeout: int = None,
)
```

<ParamField path="name" type="str" required>
  Block name

## Class Methods

### create

Creates a new block in the repository.

```python
@classmethod
def create(
    name: str,
    block_type: str,
    repo_path: str,
    color: str = None,
    configuration: dict = None,
    extension_uuid: str = None,
    language: str = None,
    pipeline: Pipeline = None,
    priority: int = None,
    replicated_block: str = None,
    require_unique_name: bool = False,
    upstream_block_uuids: list[str] = None,
    downstream_block_uuids: list[str] = None,
    config: dict = None,
    widget: bool = False,
) -> Block
```

<ParamField path="name" type="str" required>
  Block name (will be converted to snake_case for UUID)
</ParamField>

<ParamField path="block_type" type="str" required>
  Type of block to create
</ParamField>

<ParamField path="repo_path" type="str" required>
  Path to the repository
</ParamField>

<ParamField path="language" type="str" default="python">
  Programming language for the block
</ParamField>

<ParamField path="pipeline" type="Pipeline">
  Pipeline to add the block to
</ParamField>

<ParamField path="upstream_block_uuids" type="list[str]">
  List of upstream block dependencies
</ParamField>

<ResponseField name="return" type="Block">
  Newly created Block instance
</ResponseField>

**Example:**

```python
from mage_ai.data_preparation.models.block import Block
from mage_ai.data_preparation.models.pipeline import Pipeline

pipeline = Pipeline.get('my_pipeline')

block = Block.create(
    'Load Customer Data',
    'data_loader',
    repo_path=pipeline.repo_path,
    language='python',
    pipeline=pipeline
)
```

## Instance Methods

### execute_sync

Executes the block synchronously.

```python
def execute_sync(
    analyze_outputs: bool = False,
    block_run_outputs_cache: dict = None,
    build_block_output_stdout: Callable = None,
    custom_code: str = None,
    execution_partition: str = None,
    from_notebook: bool = False,
    global_vars: dict = None,
    logger: Logger = None,
    logging_tags: dict = None,
    run_all_blocks: bool = False,
    update_status: bool = True,
    store_variables: bool = True,
    verify_output: bool = True,
    runtime_arguments: dict = None,
    dynamic_block_index: int = None,
) -> dict
```

<ParamField path="global_vars" type="dict">
  Global variables available to the block during execution
</ParamField>

<ParamField path="execution_partition" type="str">
  Partition identifier for variable storage
</ParamField>

<ParamField path="from_notebook" type="bool" default="False">
  Whether execution is from the notebook interface
</ParamField>

<ParamField path="store_variables" type="bool" default="True">
  Whether to persist output variables to disk
</ParamField>

<ParamField path="runtime_arguments" type="dict">
  Additional runtime arguments
</ParamField>

<ResponseField name="return" type="dict">
  Dictionary containing execution output with key 'output'
</ResponseField>

**Example:**

```python
block = pipeline.get_block('load_data')

result = block.execute_sync(
    global_vars={'env': 'production', 'date': '2024-01-01'},
    store_variables=True,
    run_all_blocks=False
)

print(f"Execution result: {result}")
```

### async execute

Executes the block asynchronously.

```python

    analyze_outputs: bool = False,
    build_block_output_stdout: Callable = None,
    custom_code: str = None,
    global_vars: dict = None,
    run_all_blocks: bool = False,
    run_sensors: bool = True,
    update_status: bool = True,
    parallel: bool = True,

```

```python
        parallel=True

asyncio.run(run_block())
```

## Complete Example

```python
from mage_ai.data_preparation.models.pipeline import Pipeline
from mage_ai.data_preparation.models.block import Block
import pandas as pd

# Get or create pipeline
pipeline = Pipeline.get('customer_etl')

# Create data loader block
loader = Block.create(
    'load_customers',
    'data_loader',
    repo_path=pipeline.repo_path,
    language='python',
    pipeline=pipeline
)

# Update block content
loader_code = '''
@data_loader
def load_data(*args, **kwargs):
    # Load data from API
    return pd.DataFrame({
        'customer_id': [1, 2, 3],
        'name': ['Alice', 'Bob', 'Charlie']
    })
'''
loader.update_content(loader_code)

# Create transformer block
transformer = Block.create(
    'clean_data',
    'transformer',
    repo_path=pipeline.repo_path,
    language='python',
    pipeline=pipeline,
    upstream_block_uuids=['load_customers']
)

transformer_code = '''
@transformer
def transform(data, *args, **kwargs):
    # Clean and transform data
    data['name'] = data['name'].str.upper()
    return data
'''
transformer.update_content(transformer_code)

# Execute blocks
loader_result = loader.execute_sync(
    global_vars={'env': 'production'},
    store_variables=True
)

transformer_result = transformer.execute_sync(
    global_vars={'env': 'production'},
    store_variables=True
)

print(f"Pipeline completed with {len(pipeline.blocks_by_uuid)} blocks")
```
