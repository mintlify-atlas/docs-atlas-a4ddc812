### BigQuery

Load data into Google BigQuery datasets with automatic schema management.

**Configuration:**

```json
{
  "project_id": "my-gcp-project",
  "dataset": "raw_data",
  "path_to_credentials_json_file": "/path/to/credentials.json",
  "location": "US"
}
```

**Advanced Options:**

```json
{
  "project_id": "my-gcp-project",
  "dataset": "analytics",
  "credentials_info": {
    "type": "service_account",
    "project_id": "my-project",
    "private_key_id": "key-id",
    "private_key": "-----BEGIN PRIVATE KEY-----\\n..."
  },
  "use_batch_load": true,
  "disable_update_column_types": false
}
```

**Features:**
- Automatic table creation and schema updates
- Batch loading with Parquet/CSV formats
- Partition support based on date columns
- Upsert operations with unique constraints
- Handles JSON and ARRAY column types

**Example:**

```python
from mage_integrations.destinations.bigquery import BigQuery

destination = BigQuery(
    config={
        'project_id': 'my-project',
        'dataset': 'raw_data',
        'path_to_credentials_json_file': './credentials.json',
    }
)

# Process records
destination.process_record(
    stream='users',
    schema=schema,
    record={
        'id': 1,
        'name': 'John Doe',
        'created_at': '2024-01-01T00:00:00Z',
    }
)
```

### Snowflake

Write data to Snowflake with support for various authentication methods.

**Configuration:**

```json
{
  "account": "xy12345.us-east-1",
  "warehouse": "LOADING_WH",
  "database": "ANALYTICS",
  "schema": "RAW_DATA",
  "username": "MAGE_LOADER",
  "password": "secure_password",
  "role": "LOADER"
}
```

**Private Key Authentication:**

```json
{
  "account": "xy12345.us-east-1",
  "warehouse": "LOADING_WH",
  "database": "ANALYTICS",
  "schema": "RAW_DATA",
  "username": "MAGE_LOADER",
  "private_key_file": "/path/to/rsa_key.p8",
  "private_key_file_pwd": "key_password"
}
```

**Features:**
- Auto-creates tables and updates schemas
- Supports VARIANT type for JSON data
- Efficient bulk loading
- Handle updates with MERGE statements

## Common Configuration Options

### Unique Constraints & Upserts

Define how to handle duplicate records:

```python
config = {
    'unique_constraints': ['id'],  # Columns that define uniqueness
    'unique_conflict_method': 'UPDATE',  # UPDATE or IGNORE
}
```

### Partition Keys

Optimize warehouse performance with partitioning:

```python
config = {
    'partition_keys': ['created_date'],  # Partition by date column
}
```

### Type Mapping

Customize how data types are converted:

```python
config = {
    'disable_update_column_types': False,  # Allow schema updates
    'use_lowercase': True,  # Lowercase column names
}
```

## Installation

Install destination-specific dependencies:

```bash
# BigQuery
pip install "mage-ai[bigquery]"

# Snowflake
pip install "mage-ai[snowflake]"

# PostgreSQL
pip install "mage-ai[postgres]"

# Streaming (Kafka, etc.)
pip install "mage-ai[streaming]"

# All destinations
pip install "mage-ai[all]"
```
